# Welcome to AngryGuy

This is the first post on the AngryGuy Hugo site. Hugo has been successfully installed and configured!

## What is Hugo?

Hugo is a fast and flexible static site generator written in Go. It's perfect for blogs, documentation, portfolios, and more.

## Getting Started

To run this site locally:

```bash
hugo server -D
```

Then visit http://localhost:1313 in your browser.
